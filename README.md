# TheSuit

TheSuit is a modern all-in-one business operating system that helps companies manage HR, payroll, attendance, recruitment, employee data, and workplace operations from a single platform.

![TheSuit Preview](./screenshots/preview.png)

---

## ✨ Features

- Employee Management
- Payroll Processing
- Attendance Tracking
- Leave Management
- Recruitment System
- Analytics Dashboard
- Notifications & Alerts
- Role-Based Access Control
- Modern Responsive UI
- REST API Integration

---

## 🛠 Tech Stack

### Frontend
- React
- TypeScript
- TailwindCSS
- React Router
- Axios
- ShadCN UI

### Backend
- Django
- Django REST Framework
- PostgreSQL
- Redis
- Celery

### DevOps
- AWS EC2
- Nginx
- Gunicorn
- Docker
